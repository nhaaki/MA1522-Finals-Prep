<Section id="s6-1" chapterId="ch6" meta="§ 6.1" title="Eigenvalues & Eigenvectors">

### Definition

<Defn>A real number $\lambda$ is an **eigenvalue** of $A$ if there exists a **nonzero** vector $v$ (the **eigenvector**) such that $Av = \lambda v$.</Defn>

Equivalently, $(\lambda I - A)v = \mathbf{0}$ has a nontrivial solution, i.e., $\lambda I - A$ is **singular**.

### Characteristic Polynomial

$$\text{char}(A) = \det(\lambda I - A)$$

$\lambda$ is an eigenvalue $\Leftrightarrow$ $\lambda$ is a **root** of $\det(\lambda I - A) = 0$.

### Algorithm: Finding Eigenvalues and Eigenvectors

1. Compute $\det(\lambda I - A)$ — the characteristic polynomial.
2. Find the roots of $\det(\lambda I - A) = 0$.
3. For each eigenvalue $\lambda$, find the **eigenspace** $E_\lambda = \text{Null}(\lambda I - A)$: solve $(\lambda I - A)x = \mathbf{0}$.

### Multiplicities

- **Algebraic multiplicity** $r_\lambda$: the power of $(\lambda - \mu)$ in the characteristic polynomial.
- **Geometric multiplicity**: $\dim(E_\lambda) = \text{nullity}(\lambda I - A)$.
- Always: $1 \leq \dim(E_\lambda) \leq r_\lambda$.

<Callout tag="Key fact">Eigenvalues of a **triangular matrix** are its diagonal entries. $A$ is invertible $\Leftrightarrow$ $0$ is **not** an eigenvalue of $A$.</Callout>

<Example title="2×2 eigenvalue computation">

$$A = \begin{bmatrix}3 & 1 \\ 1 & 3\end{bmatrix}$$

$$\det(\lambda I - A) = \det\begin{bmatrix}\lambda-3 & -1 \\ -1 & \lambda-3\end{bmatrix} = (\lambda-3)^2 - 1 = \lambda^2 - 6\lambda + 8 = (\lambda-4)(\lambda-2)$$

Eigenvalues: $\lambda_1 = 4$, $\lambda_2 = 2$.

For $\lambda_1 = 4$: $(4I - A) = \begin{bmatrix}1&-1\\-1&1\end{bmatrix} \to \begin{bmatrix}1&-1\\0&0\end{bmatrix}$, so $E_4 = \text{span}\{(1,1)^T\}$.

For $\lambda_2 = 2$: $(2I - A) = \begin{bmatrix}-1&-1\\-1&-1\end{bmatrix} \to \begin{bmatrix}1&1\\0&0\end{bmatrix}$, so $E_2 = \text{span}\{(-1,1)^T\}$.

</Example>

<Section id="s6-2" chapterId="ch6" meta="§ 6.2" title="Diagonalization">

### Definition

<Defn>$A$ is **diagonalizable** if $A = PDP^{-1}$ for some invertible $P$ and diagonal $D$.</Defn>

The columns of $P$ are eigenvectors of $A$; the diagonal entries of $D$ are the corresponding eigenvalues.

### Key Theorem

$A$ is diagonalizable $\Leftrightarrow$ $A$ has **$n$ linearly independent eigenvectors**.

### Conditions for Diagonalizability

$A$ is diagonalizable iff:
1. The characteristic polynomial **splits into linear factors** over $\mathbb{R}$, **and**
2. For every eigenvalue $\lambda$: $\dim(E_\lambda) = r_\lambda$ (geometric multiplicity = algebraic multiplicity).

### Algorithm for Diagonalization

1. Find all eigenvalues (roots of char. poly.).
2. For each $\lambda_i$, find a basis for $E_{\lambda_i}$.
3. Check $\dim(E_{\lambda_i}) = r_{\lambda_i}$ for all $i$.
4. $P = (v_1 \mid \cdots \mid v_n)$ (eigenvectors as columns), $D = \text{diag}(\lambda_1, \ldots, \lambda_n)$.

### Powers of Diagonalizable Matrices

$$A^k = P D^k P^{-1}, \qquad D^k = \text{diag}(\lambda_1^k, \ldots, \lambda_n^k)$$

<Example title="Diagonalizing the matrix from §6.1">

From §6.1: $A = \begin{bmatrix}3&1\\1&3\end{bmatrix}$, eigenvalues $4$ and $2$.

$$P = \begin{bmatrix}1 & -1 \\ 1 & 1\end{bmatrix}, \quad D = \begin{bmatrix}4 & 0 \\ 0 & 2\end{bmatrix}$$

$$P^{-1} = \frac{1}{2}\begin{bmatrix}1 & 1 \\ -1 & 1\end{bmatrix}$$

Check: $PDP^{-1} = \begin{bmatrix}1&-1\\1&1\end{bmatrix}\begin{bmatrix}4&0\\0&2\end{bmatrix}\frac{1}{2}\begin{bmatrix}1&1\\-1&1\end{bmatrix} = A$ ✓

</Example>

</Section>

<Section id="s6-3" chapterId="ch6" meta="§ 6.3" title="Orthogonal Diagonalization">

### Definition

<Defn>$A$ is **orthogonally diagonalizable** if $A = PDP^T$ for some **orthogonal** matrix $P$ and diagonal $D$.</Defn>

### The Spectral Theorem

$$A \text{ is orthogonally diagonalizable} \Leftrightarrow A \text{ is symmetric } (A^T = A)$$

*(Over $\mathbb{R}$; this course works over the reals.)*

### Key Theorem

Eigenspaces of a symmetric matrix corresponding to **distinct eigenvalues** are **orthogonal** to each other.

### Algorithm for Orthogonal Diagonalization

1. Compute the characteristic polynomial of the symmetric matrix $A$.
2. Find a basis for each eigenspace.
3. Apply **Gram-Schmidt** within each eigenspace to get an orthonormal basis.
4. Combine all orthonormal eigenvectors to form $P$; eigenvalues on diagonal of $D$.
5. Then $A = PDP^T$.

<Callout tag="Note">For a symmetric matrix, eigenvectors from **different** eigenspaces are automatically orthogonal — you only need Gram-Schmidt within each eigenspace when the eigenspace is multi-dimensional.</Callout>

</Section>

<Section id="s6-4" chapterId="ch6" meta="§ 6.4" title="Application: Markov Chains">

### Markov Chains

- A **probability vector**: nonnegative entries summing to 1.
- A **stochastic matrix**: square matrix whose **columns** are probability vectors.
- A **Markov chain**: sequence $x_0, x_1 = Px_0, x_2 = Px_1, \ldots$ tracking state distributions.
- A **steady-state (equilibrium) vector**: a probability vector $v$ with $Pv = v$ (eigenvector for eigenvalue 1).

### Algorithm: Finding the Equilibrium Vector

1. Solve $(I - P)x = \mathbf{0}$ for an eigenvector $u$ with eigenvalue 1.
2. Normalize: $v = \dfrac{u}{\sum_i u_i}$ so the entries sum to 1.

### Regular Stochastic Matrices

$P$ is **regular** if some power $P^k$ has all positive entries. For a regular $P$, the Markov chain converges to a **unique** equilibrium vector regardless of initial state.

### Google PageRank

1. Build adjacency matrix $A$ from the web link structure.
2. Normalize columns to get a stochastic transition matrix $P$.
3. Find the steady-state vector — this is the PageRank.

<Example title="2-state Markov chain">

$$P = \begin{bmatrix}0.9 & 0.2 \\ 0.1 & 0.8\end{bmatrix}$$

Solve $(I-P)x = 0$: $\begin{bmatrix}0.1 & -0.2\\ -0.1 & 0.2\end{bmatrix}x = 0$, giving $x_1 = 2x_2$.

Take $u = (2,1)^T$. Normalize: $v = \left(\tfrac{2}{3}, \tfrac{1}{3}\right)^T$. The chain converges to state 1 with probability $\tfrac{2}{3}$.

</Example>

</Section>

<Section id="s6-5" chapterId="ch6" meta="§ 6.5" title="Application: Singular Value Decomposition (SVD)">

### SVD Theorem

Every $m \times n$ matrix $A$ can be written as

$$A = U \Sigma V^T$$

where $U$ ($m \times m$) and $V$ ($n \times n$) are **orthogonal**, and $\Sigma$ ($m \times n$) has diagonal block $D = \text{diag}(\sigma_1, \ldots, \sigma_r)$ with $r = \text{rank}(A)$.

### Singular Values

$\sigma_i = \sqrt{\mu_i}$, where $\mu_1 \geq \mu_2 \geq \cdots \geq \mu_r > 0$ are the **nonzero eigenvalues** of $A^T A$.

### Algorithm

1. Find eigenvalues $\mu_1 \geq \cdots \geq \mu_n \geq 0$ of $A^T A$ (which is symmetric, so orthogonally diagonalizable).
2. Set $\sigma_i = \sqrt{\mu_i}$ for $i = 1, \ldots, r$. Build $\Sigma$.
3. Find orthonormal eigenvectors $v_1, \ldots, v_n$ of $A^T A$; set $V = (v_1 \mid \cdots \mid v_n)$.
4. For $i \leq r$: $u_i = \dfrac{1}{\sigma_i} A v_i$ (orthonormal basis for $\text{Col}(A)$).
5. Extend $\{u_1, \ldots, u_r\}$ to an orthonormal basis $\{u_1, \ldots, u_m\}$ for $\mathbb{R}^m$.
6. Set $U = (u_1 \mid \cdots \mid u_m)$.

### Notes

- $\text{rank}(A) = r$ = number of positive singular values.
- For symmetric $A$: singular values $= |\lambda_i|$ (absolute values of eigenvalues).
- SVD is the foundation of **principal component analysis (PCA)**, **image compression**, and **low-rank approximation**.

</Section>
