<Section id="s2-1" chapterId="ch2" meta="§ 2.1" title="Definition & Special Types of Matrices">

### Definition

A **matrix** $A = (a_{ij})_{m \times n}$ is a rectangular array of real numbers with $m$ rows and $n$ columns. The entry $a_{ij}$ is in the $i$-th row and $j$-th column.

### Special Types

| Type | Description |
|---|---|
| **Square matrix** | $m = n$ |
| **Zero matrix** $\mathbf{0}$ | All entries are 0 |
| **Identity matrix** $I_n$ | Square; 1s on diagonal, 0s elsewhere |
| **Diagonal matrix** | Square; all off-diagonal entries are 0 |
| **Upper triangular** | All entries **below** main diagonal are 0 |
| **Lower triangular** | All entries **above** main diagonal are 0 |
| **Symmetric matrix** | $A^T = A$ |

<Callout tag="Think about">Is a symmetric upper triangular matrix necessarily a diagonal matrix? (Yes — the only way a matrix can be both upper triangular and its own transpose is if all off-diagonal entries are 0.)</Callout>

</Section>

<Section id="s2-2" chapterId="ch2" meta="§ 2.2" title="Matrix Algebra">

### Matrix Addition & Scalar Multiplication

- **Scalar multiplication**: multiply every entry by $c$.
- **Addition**: add entry-by-entry (matrices must have the same size).

Key properties: commutative, associative, distributive, additive identity $\mathbf{0}$, additive inverse $-A$.

### Matrix Multiplication

If $A$ is $m \times p$ and $B$ is $p \times n$, then $C = AB$ is $m \times n$ with

$$C_{ij} = \sum_{k=1}^{p} a_{ik} b_{kj}$$

<Callout tag="Caution" variant="warn">Matrix multiplication is **not** commutative: $AB \neq BA$ in general. Also, $AB = \mathbf{0}$ does **not** imply $A = \mathbf{0}$ or $B = \mathbf{0}$.</Callout>

<Example title="2×2 product">

$$\begin{bmatrix} 1 & 2 \\ 3 & 4 \end{bmatrix} \begin{bmatrix} 5 & 6 \\ 7 & 8 \end{bmatrix} = \begin{bmatrix} 1\cdot5+2\cdot7 & 1\cdot6+2\cdot8 \\ 3\cdot5+4\cdot7 & 3\cdot6+4\cdot8 \end{bmatrix} = \begin{bmatrix} 19 & 22 \\ 43 & 50 \end{bmatrix}$$

</Example>

### Transpose

$(A^T)_{ij} = A_{ji}$. Key properties:

- $(A^T)^T = A$
- $(cA)^T = cA^T$
- $(A+B)^T = A^T + B^T$
- $(AB)^T = B^T A^T$

A square matrix is **symmetric** if $A^T = A$.

</Section>

<Section id="s2-3" chapterId="ch2" meta="§ 2.3" title="Linear System & Matrix Equation">

### Matrix Equation Form

$Ax = b$: coefficient matrix $A$, variable column vector $x$, constant column vector $b$.

### Vector Equation Form

$$x_1 a_1 + x_2 a_2 + \cdots + x_n a_n = b$$

where $a_1, \ldots, a_n$ are the **column vectors** of $A$.

### Homogeneous Systems

$Ax = \mathbf{0}$ always has the **trivial solution** $x = \mathbf{0}$. A nonzero solution is a **nontrivial solution**. The system has infinitely many solutions $\Leftrightarrow$ nontrivial solutions exist.

### Key Lemma

If $v$ is a particular solution to $Ax = b$ and $u$ satisfies $Au = \mathbf{0}$, then $v + u$ is also a solution to $Ax = b$.

$$\text{General solution to } Ax = b \;=\; (\text{particular solution}) + (\text{general solution to } Ax = \mathbf{0})$$

### Block Multiplication

$$AB = A\begin{pmatrix}b_1 \mid b_2 \mid \cdots \mid b_n\end{pmatrix} = \begin{pmatrix}Ab_1 \mid Ab_2 \mid \cdots \mid Ab_n\end{pmatrix}$$

Multiple systems with the same coefficient matrix can be solved simultaneously by forming a combined augmented matrix $[A \mid b_1 \mid b_2 \mid \cdots]$.

</Section>

<Section id="s2-4" chapterId="ch2" meta="§ 2.4" title="Inverse of Matrices">

### Definition

An $n \times n$ matrix $A$ is **invertible** if there exists $B$ such that $AB = I_n = BA$. The inverse $B = A^{-1}$ is **unique**. A non-invertible square matrix is called **singular**.

### Inverse of a 2×2 Matrix

For $A = \begin{pmatrix}a & b \\ c & d\end{pmatrix}$, $A$ is invertible $\Leftrightarrow$ $ad - bc \neq 0$, and

$$A^{-1} = \frac{1}{ad - bc}\begin{pmatrix}d & -b \\ -c & a\end{pmatrix}$$

<Example title="2×2 inverse">

$$A = \begin{bmatrix} 3 & 1 \\ 5 & 2 \end{bmatrix}, \quad \det(A) = 3\cdot2 - 1\cdot5 = 1$$

$$A^{-1} = \frac{1}{1}\begin{bmatrix} 2 & -1 \\ -5 & 3 \end{bmatrix} = \begin{bmatrix} 2 & -1 \\ -5 & 3 \end{bmatrix}$$

Check: $AA^{-1} = \begin{bmatrix}3&1\\5&2\end{bmatrix}\begin{bmatrix}2&-1\\-5&3\end{bmatrix} = \begin{bmatrix}1&0\\0&1\end{bmatrix}$ ✓

</Example>

### Algorithm to Compute $A^{-1}$

Form the augmented matrix $(A \mid I_n)$ and reduce to RREF:

$$(A \mid I_n) \xrightarrow{\text{RREF}} (I_n \mid A^{-1})$$

If the left block does **not** reduce to $I_n$, then $A$ is **not invertible**.

### Properties of Inverses

- $(A^{-1})^{-1} = A$
- $(aA)^{-1} = \frac{1}{a}A^{-1}$
- $(A^T)^{-1} = (A^{-1})^T$
- $(AB)^{-1} = B^{-1}A^{-1}$

### Invertibility and Linear Systems

If $A$ is invertible, then $Ax = b$ has a **unique solution** $x = A^{-1}b$ for every $b$.

</Section>

<Section id="s2-5" chapterId="ch2" meta="§ 2.5" title="Elementary Matrices">

### Definition

An **elementary matrix** $E$ is obtained from the identity $I_n$ by a **single** elementary row operation.

### Key Theorem

Pre-multiplying $A$ by $E$ performs the corresponding row operation on $A$:

$$A \xrightarrow{r} EA$$

If $B = E_k \cdots E_2 E_1 A$, then $A$ and $B$ are row-equivalent.

### Inverses of Elementary Matrices

| Row operation on $I$ | Elementary matrix inverse |
|---|---|
| $R_i + cR_j$ | $R_i - cR_j$ |
| $R_i \leftrightarrow R_j$ | $R_i \leftrightarrow R_j$ (itself) |
| $cR_i$ | $\frac{1}{c}R_i$ |

Every elementary matrix is **invertible**.

</Section>

<Section id="s2-6" chapterId="ch2" meta="§ 2.6" title="Equivalent Statements for Invertibility">

For a square matrix $A$ of order $n$, the following are **all equivalent**:

1. $A$ is invertible.
2. $A^T$ is invertible.
3. $A$ has a **left inverse** ($BA = I$).
4. $A$ has a **right inverse** ($AB = I$).
5. RREF of $A$ is $I_n$.
6. $A$ is a product of elementary matrices.
7. $Ax = \mathbf{0}$ has only the trivial solution.
8. $Ax = b$ is consistent for every $b \in \mathbb{R}^n$.

*(More equivalences are added in Chapters 3, 4, 6, 7 — see the Master Theorem in the Appendix.)*

</Section>

<Section id="s2-7" chapterId="ch2" meta="§ 2.7" title="LU Factorization">

### Definition

An **LU factorization** of $A$ writes $A = LU$ where:
- $L$ is **unit lower triangular** (lower triangular with 1s on the diagonal).
- $U$ is a **row-echelon form** of $A$.

<Callout tag="Note" variant="warn">Not every matrix has an LU factorization. It requires that the row reduction of $A$ to REF can be done **without any row swaps**.</Callout>

### Algorithm

Use only row operations of the form $R_i + cR_j$ ($i > j$, adding a multiple of a higher row to a lower one) to reduce $A$ to REF $U$. The matrix $L$ is formed from the negative of the multipliers used:

$$L = \begin{bmatrix} 1 & & \\ m_{21} & 1 & \\ m_{31} & m_{32} & 1 \end{bmatrix}$$

where $m_{ij}$ is the multiplier used to zero out the $(i,j)$ entry.

### Solving $Ax = b$ via LU

Given $Ax = b$ and $A = LU$:
1. Solve $Ly = b$ (forward substitution, top to bottom).
2. Solve $Ux = y$ (back substitution, bottom to top).

<Example title="3×3 LU factorization">

$$A = \begin{bmatrix} 2 & 4 & -2 \\ 4 & 9 & -3 \\ -2 & -3 & 7 \end{bmatrix}$$

$R_2 - 2R_1$, $R_3 + R_1$, then $R_3 + R_2/3$... gives:

$$L = \begin{bmatrix} 1 & 0 & 0 \\ 2 & 1 & 0 \\ -1 & 1 & 1 \end{bmatrix}, \quad U = \begin{bmatrix} 2 & 4 & -2 \\ 0 & 1 & 1 \\ 0 & 0 & 4 \end{bmatrix}$$

</Example>

</Section>

<Section id="s2-8" chapterId="ch2" meta="§ 2.8" title="Determinant by Cofactor Expansion">

### Cofactors

The **(i,j)-cofactor** of $A$ is $A_{ij} = (-1)^{i+j} \det(M_{ij})$, where $M_{ij}$ is the matrix obtained by deleting row $i$ and column $j$.

### Cofactor Expansion

**Along row $i$:**

$$\det(A) = \sum_{k=1}^n a_{ik} A_{ik} = a_{i1}A_{i1} + a_{i2}A_{i2} + \cdots + a_{in}A_{in}$$

**Along column $j$:** same formula with $a_{kj} A_{kj}$.

<Callout tag="Tip">Expand along the row or column with the most zeros to minimize computation.</Callout>

### Key Results

- $\det(A) = \det(A^T)$
- For a **triangular matrix**: $\det(A) = a_{11} a_{22} \cdots a_{nn}$ (product of diagonal entries)
- If $A$ has a zero row or column, then $\det(A) = 0$

<Example title="3×3 cofactor expansion along row 1">

$$A = \begin{bmatrix} 1 & 2 & 0 \\ 3 & 1 & 4 \\ 0 & -1 & 2 \end{bmatrix}$$

$$\det(A) = 1\cdot\det\begin{bmatrix}1&4\\-1&2\end{bmatrix} - 2\cdot\det\begin{bmatrix}3&4\\0&2\end{bmatrix} + 0$$

$$= 1\cdot(2+4) - 2\cdot(6-0) = 6 - 12 = -6$$

</Example>

</Section>

<Section id="s2-9" chapterId="ch2" meta="§ 2.9" title="Determinant by Reduction">

### Effect of Row Operations on $\det$

| Row operation | Effect on $\det$ |
|---|---|
| $R_i + aR_j$ (add multiple) | $\det(B) = \det(A)$ (unchanged) |
| $cR_i$ (scale row) | $\det(B) = c\cdot\det(A)$ |
| $R_i \leftrightarrow R_j$ (swap) | $\det(B) = -\det(A)$ |

### Algorithm

Reduce $A$ to upper-triangular form (REF) $U$ using elementary row operations. Track the effect on the determinant. Since $\det(U) = u_{11} u_{22} \cdots u_{nn}$ (product of diagonal entries), we have:

$$\det(A) = \frac{u_{11} u_{22} \cdots u_{nn}}{\prod_k \det(E_k)}$$

where $E_k$ are the elementary matrices applied. In practice: keep a running scalar $d$ starting at 1, multiply by $-1$ for each swap and by $1/c$ for each scale, then $\det(A) = d \cdot u_{11} \cdots u_{nn}$.

<Callout tag="Key fact">If $A$ has two equal rows (or columns), $\det(A) = 0$. Swapping them leaves $A$ unchanged but negates $\det$, so $\det(A) = -\det(A) \Rightarrow \det(A) = 0$.</Callout>

</Section>

<Section id="s2-10" chapterId="ch2" meta="§ 2.10" title="Properties of Determinant">

### Key Theorems

$$\det(AB) = \det(A)\det(B)$$

$$\det(A^{-1}) = \det(A)^{-1} \quad (A \text{ invertible})$$

$$\det(cA) = c^n \det(A) \quad (A \text{ is } n\times n)$$

### Adjoint and Inverse Formula

The **adjoint** $\text{adj}(A)$ is the matrix whose $(i,j)$-entry is the $(j,i)$-cofactor of $A$ (i.e., the transpose of the cofactor matrix).

$$A \cdot \text{adj}(A) = \det(A) \cdot I$$

$$A^{-1} = \frac{1}{\det(A)} \text{adj}(A) \quad (\text{if } A \text{ invertible})$$

### Equivalent Statements (Updated)

Add to the list from §2.6: **$\det(A) \neq 0 \Leftrightarrow A$ is invertible.**

### Cramer's Rule

For $Ax = b$ with $A$ invertible, the $i$-th component of the solution is:

$$x_i = \frac{\det(A_i)}{\det(A)}$$

where $A_i$ is $A$ with column $i$ replaced by $b$.

<Example title="Cramer's rule on a 2×2 system">

$$\begin{cases} 3x_1 + x_2 = 5 \\ 5x_1 + 2x_2 = 8 \end{cases}, \quad A = \begin{bmatrix}3&1\\5&2\end{bmatrix}, \quad b = \begin{bmatrix}5\\8\end{bmatrix}$$

$$\det(A) = 1, \quad A_1 = \begin{bmatrix}5&1\\8&2\end{bmatrix},\quad A_2 = \begin{bmatrix}3&5\\5&8\end{bmatrix}$$

$$x_1 = \frac{10-8}{1} = 2, \quad x_2 = \frac{24-25}{1} = -1$$

</Example>

</Section>
